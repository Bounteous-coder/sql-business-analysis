# Jira Story Starters

## Story 1: Build KPI Baseline Query

- As a data analyst, I want a KPI baseline so stakeholders can see current business health.
- Acceptance criteria:
  - Query returns customer count, order count, revenue, average order value.
  - Query is saved and reviewed.

## Story 2: Segment Revenue Analysis

- As a data analyst, I want segment-level performance so marketing can prioritize spend.
- Acceptance criteria:
  - Segment-level revenue and average order value are calculated.
  - At least one recommendation is documented.

## Story 3: Monthly Trend Analysis

- As a data analyst, I want month-over-month trends so leadership can detect growth or slowdown.
- Acceptance criteria:
  - Monthly orders and revenue displayed in time order.
  - Trend interpretation added to executive summary.
